Asyncronous actions are the actions that we initiate now and they finish lates. e.g. setTimeout()
Syncronous actions are the actions that initiates and finish one by one.
Callback functions are the functions passed into another function as an argument,which is then invoked inside the outer function to complete an action.
The solution to the callback hell are Promises.The syntax of the Promise is:-
let promise=new promise(function(resolve,reject){
//executor
});
resolve and reject are predefined in JS engine.These are two callbacks provided by the JS itself.They are called like this:-
resolve(value)->If the job is finished successfully 
reject(error)->If the job fails 
promise.finally(()=>{}) is used to perform genral cleanups.It close all the files that you opened.
Promise Chaining :- We can chain promises and make them pass the resiolve values to one another like this:-
promise.then(function(result)=>{
    alert(result);
    return promise2;
}).then...
The idea is to passthe result through the chain of .then handlers.
Attaching multiple handlers:- We can attach multiple handlers to one promise.They don't pass the result to each other;instead they process it independently.
let p is a promise
p.then(handler1)
p.then(handler2)
p.then(handler3)
all runs independently

Promise API:- There are 6 static methods of promise class:-
1.Promise.all(promises)->waits for all promises to resolve and returns the array of this results.If any one fails, it becomes the error and all other results are ignored.
2.Promise.allSettled(Promises)->waits for all the promises to settle and returns their results as an array of objects with status and value.
3.promise.race(Promises)->waits for the first promise to settle and its result/error becomes the outcome.
4.Promise.any(Promises)->waits for the first promise to fulfil (and not rejected), and its result becomes the outcome. Throws Aggregate Eror if all the promises are rejected.
5.Promise.resolve(value)->makes a resolved promise with the given value.
6.Promise.reject(error)->makes a rejected promise with the given error.
